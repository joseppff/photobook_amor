# Backend Images

Coloca las imágenes de tu photobook en las carpetas correspondientes por año:

```
backend/images/
├── 2024/
│   ├── 01-01-2024.jpg
│   ├── 15-02-2024.jpg
│   └── 20-03-2024.jpg
├── 2025/
│   ├── 10-01-2025.jpg
│   └── 14-02-2025.jpg
└── 2026/
    └── (futuras fotos)
```

## Formato de nombres de archivo

Usa el formato: `dd-mm-yyyy.jpg` para que coincida con las fechas en `photobook.json`

## Formatos soportados

- `.jpg` / `.jpeg`
- `.png`
- `.webp`
